# Markdownメモ閲覧GUIアプリ

本アプリは、サブモニターでの表示用に適した、シンプルで軽量なMarkdown（.md）メモ閲覧GUIアプリです。特定のディレクトリ内にあるMarkdownファイルを自動で探索し、フォルダーの階層構造を保持したまま左側のサイドバーに一覧表示します。

## 主な機能

* **階層構造の自動反映**: 指定したディレクトリ内のフォルダー構成がそのままアプリ内の階層メニューとして表示されます。
* **Markdown要素の描画**: 箇条書きリスト（`*`, `-`, `+`）、番号付きリスト、水平線に対応しています。
* **自動改行**: ウィンドウの大きさを検知し、テキストを自動で改行します。
* **コードブロック表示**: 複数行コードの読み込みに対応し、ワンクリックでコードをクリップボードにコピーできるボタンが自動で追加されます。
* **画像表示**: 相対パスまたは絶対パスでの画像読み込みに対応し、ウィンドウ幅からはみ出す場合は自動で縮小されます。
* **HTML表示機能**: 複雑なMarkdownファイルの場合、内部でHTMLに一度変換して表示させることができます。

## 必要ライブラリの導入

実行にはPython環境が必要です。事前に以下のライブラリをターミナル（コマンドプロンプト）からインストールしてください。

```bash
pip install customtkinter pyperclip markdown pillow tkhtmlview
```

## 使い方

1. 任意のファイル名（例: `main.py`）でソースコードを保存します。
2. アプリと同じ階層にメモ用のフォルダー（デフォルト設定では `data` フォルダー）を作成し、中にMarkdownファイル（`.md`）を配置します。
3. 以下のコマンドを実行してGUIを起動します。

```bash
python main.py
```

## 設定の変更

ソースコード冒頭の大域変数を書き換えることで、フォントや検索ディレクトリを簡単に変更できます。

* `PATH`: 検索対象とするディレクトリのパス（初期値: `"./data"`）
* `FONT` / `FONTSIZE`: 左側メニューのフォントとサイズ
* `MDFONT` / `MDFONTSIZE`: 本文表示エリアのフォントとサイズ
* `MDCODEFONT`: コードブロック用のフォント
* `EXCEPTION_FILE_NAME`: 検索から除外したいフォルダー名
